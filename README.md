## 简洁

&emsp;&emsp;符合 [Promise/A+](https://promisesaplus.com/) 规范的手写版`Promise`，以及相关测试套件，支持多环境。

* [手写指南](https://juejin.cn/post/7205549412352376892)

## 测试

### 安装依赖

```sh
npm i
```

### Node 环境

```sh
npm run test:node
```

### 默认浏览器环境

```sh
npm run test:browser
```
